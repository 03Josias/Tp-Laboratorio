# Tp-Laboratorio
Sistema de Gestión para Gimnasio (C++) 
  Sistema de Gestión de Gimnasio
Este es un proyecto de gestión integral desarrollado en C++ diseñado para facilitar la administración de clientes, asistencias, pagos y la generación de reportes en un entorno de gimnasio.

  Funcionalidades Clave
El sistema está estructurado con un menú principal que ofrece acceso a las siguientes áreas funcionales:
-Gestión de Clientes y Asistencias: Permite administrar la información de los clientes, registrar sus asistencias y consultar listados por mes, año o disciplina.
-Gestión Financiera (Recaudación): Permite registrar los pagos realizados por los clientes y generar reportes detallados de la recaudación anual o por disciplina.
-Informes y Consultas: Ofrece diversas salidas esenciales para el análisis y la toma de decisiones, como listados de clientes (activos/inactivos), consultas por -ID/Nombre/DNI, y el cálculo de la edad promedio por discipli
-Ranking y Dashboard: Muestra un resumen de información clave en el Dashboard (clientes activos, recaudación del mes) y genera un Ranking de los clientes con más asistencias.
-Mantenimiento del Sistema: Incluye opciones de Copia de Seguridad y Restauración de datos para asegurar la integridad y seguridad de la información.

Estructura y Tecnología
Lenguaje: Desarrollado completamente en C++.

-Modelo: Utiliza un diseño modular basado en un Diagrama de Clases que incluye entidades clave como Persona, Clientes, Pagos, Disciplina, Asistencia, Fecha e Identidad.
-Salidas: La aplicación genera tres tipos de salidas fundamentales para la gestión y análisis: Listados, Consultas e Informes.
